Setup

Download the source package of MyMediaLite, and open the MyMediaLite solution file src/MyMediaLite.sln in MonoDevelop. Hit Ctrl-F8 to build the complete project. In the MyMediaLite root directory, enter 

    make download_imdb
    make download_movielens
To train the initial model, execute 
    mkdir data/models
    mono --debug src/RatingPrediction/bin/Debug/RatingPrediction.exe --training-file=data/ml1m/ratings.txt --recommender=BiasedMatrixFactorization  --recommender-options="num_iter=60 bias_reg=0.001 regularization=0.045" --save-model=data/models/ml1m-bmf.model

Then you can run the GUI demo e.g. by opening src/MovieDemo/Main.cs and hitting Ctrl-F5 in MonoDevelop.
